# Employee Management System

A simple console-based Employee Management System built in Python. Manage employee records through a menu-driven terminal interface — no database, no external libraries, just pure Python.

---

## Features

- Add employees with ID, name, department, and salary
- View all employees in a formatted list
- Search employee by ID
- Update employee details
- Delete employee by ID
- Input validation — no empty fields, no duplicate IDs, salary must be a positive number

---

## How to Run

Make sure you have Python installed, then:

```bash
python employee_management.py
```

That's it. No installations needed.

---

## Menu Options

```
=== Employee Management System ===
1. Add Employee
2. View All Employees
3. Search Employee by ID
4. Update Employee
5. Delete Employee by ID
6. Exit
```

---

## Concepts Used

- Functions
- Lists and dictionaries
- Loops and conditionals
- Exception handling (`try/except`)
- User input validation
